Description

Black Economy Folding Table is a 3d model of desk that can be easily folded and stored away when not in use.

These tables are often used for events such as banquets, conferences, and parties where temporary seating arrangements are required. For home furnishings, home dinner table serving, table laying, table setting, restaurant tableware, kitchen tableware, decorative table setting, and traditional table appointments documentary or promotional projects. Black Economy Folding Table is a high quality, photo real 3d model that will enhance detail and realism to any of your rendering projects. The model has a fully textured, detailed design that allows for close-up renders, and was originally modeled in 3ds Max 2014 and rendered with V-Ray. Renders have no postprocessing!
